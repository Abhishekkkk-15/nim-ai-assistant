---

Transform the current UI into a **modern, minimal, high-signal developer interface** with:

* Clean layout

* Smooth interactions

---

## 🎨 DESIGN REQUIREMENTS

### 1. Overall Style

* Minimal, clean, developer-focused
* Dark theme first (VS Code theme compatible)
* Soft contrast, not harsh colors
* Consistent spacing system (8px grid)

---

### 2. Typography

* Match Claude Code style:

  * Clean sans-serif
  * Proper hierarchy:

    * Headers
    * Messages
    * Metadata
* Improve readability for long conversations

---

### 3. Chat UI Redesign

Each message must support:

* User vs Agent distinction
* Message grouping
* Timestamp or subtle metadata
* Streaming response rendering

---

### 4. File Preview UI (IMPORTANT)

Implement inline file previews:

* Show code blocks as:

  * Syntax highlighted
  * Scrollable
  * With file name header

Example:

```txt
📄 app.ts
------------------
<code preview>
```

---

### 5. Subtasks / Agent Steps (CRITICAL)

Implement collapsible subtask system:

Each agent response can contain steps like:

* Planning
* File edits
* Tool usage

UI requirements:

* Collapsible sections
* Expand/collapse toggle
* Step status indicators:

  * ⏳ running
  * ✅ completed
  * ❌ failed

---

### 6. Agent Status Indicator

Show real-time agent state:

* Idle
* Thinking
* Executing tools

* Animated indicator
* Status text

### 7. Sidebar Redesign

Improve:

* Layout structure
* Sections:

  * Agents

---

### 8. Input Box UX

* Shortcut support:

### 9. Animations & Microinteractions

* Collapse/expand animation
